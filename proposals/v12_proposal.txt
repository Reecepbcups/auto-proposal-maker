# Juno v12 Upgrade Proposal\n## Summary\n\nThis is a proposal to set the Juno v12 upgrade at block height 7000000. Juno v12 is an feature rich upgrade including the following new modules:\n\n- x/Oracle Module\n- x/FeeShare Module\n- x/TokenFactory Module\n- x/GlobalFee Module\n\nThe above modules have been tested both on other Networks (uume, evmos, osmosis, and gaia respectively), on the Juno Testnet (uni-5), and our own newly added test end-2-end suite.\n\nThis update also includes the following updates:\n\n- Interchain Account Controller\n- CosmWasm 0.30.0\n- IBC v4.2.0 + IBCFees\n- 20% minimum gov deposit (prevents phishing attack spam)\n\n<br>\n\n## Technical Details & Changes\n\n###
